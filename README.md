# UNDER DEVELOPMENT

## Web Application in CodeIgniter

This project contains custom templates which can be used in based in [ CodeIgniter ](https://codeigniter.com)

* Website
* Admin Panel
* Rest API

### Tools

Following application tools are used: 

* CodeIgnite 3.1.10
* Bootstrap 4.1 CSS
    * Front-end Admin Panel
    * Front-end Website
* Sass Technology
    * Front-end Admin Panel
    * Front-end Website
* Font-awesome 4.7
    * Front-end Admin Panel
    * Front-end Website
* JWT Authentication
    * REST API
